Gradients

Gradients provide information on a function's sensitivity to perturbations in the input.

Directional derivatives

The behavior of a function f:Rn→Rf:\mathbb R^n\to\mathbb Rf:Rn→R along the ray {x+αd∣α∈R+}\{x+αd\mid α\in\mathbb R_+\}{x+αd∣α∈R+​}, where xxx and ddd are nnn-vectors, is given by the univariate function

ϕ(α)=f(x+αd). \phi(\alpha) = f(x+αd). ϕ(α)=f(x+αd).

From standard calculus, the derivative of ϕ\phiϕ at the origin, when it exists, is the limit

ϕ′(0)=lim⁡α→0+ϕ(α)−ϕ(0)α. \phi'(0) = \lim_{α\to0^+}\frac{\phi(α)-\phi(0)}{α}. ϕ′(0)=α→0+lim​αϕ(α)−ϕ(0)​.

We thus arrive at the following definition.

Definition: (directional derivative) The directional derivative of a function f:Rn→Rf:\mathbb R^n\to\mathbb Rf:Rn→R at a point x∈Rnx\in\mathbb R^nx∈Rn, along a direction d∈Rnd\in\mathbb R^nd∈Rn, is the limit
f′(x;d)=lim⁡α→0+f(x+αd)−f(x)α. f'(x;d) = \lim_{α\to0^+}\frac{f(x+αd)-f(x)}{α}. f′(x;d)=α→0+lim​αf(x+αd)−f(x)​.

It follows immediately from this definition that the partial derivatives of fff are simply the directional derivatives of fff along the each of the canonical unit direction e1,…,ene_1,\ldots,e_ne1​,…,en​, i.e.,

∂f∂xi(x)≡f′(x;ei). \frac{\partial f}{\partial x_i}(x) \equiv f'(x;e_i). ∂xi​∂f​(x)≡f′(x;ei​).

Descent directions

A nonzero vector ddd is a descent direction of fff at xxx if the directional derivative is negative:

f′(x;d)<0. f'(x;d) < 0. f′(x;d)<0.

It follows directly from the definition of the directional derivative that f(x+αd)<f(x)f(x+αd) < f(x)f(x+αd)<f(x) for all positive α\alphaα small enough.

Gradient vector

The gradient of the function fff is the collection of all the partial derivatives:

∇f(x)=[∂f∂x1(x)⋮∂f∂xn(x)]. \nabla f(x) = \begin{bmatrix} \frac{\partial f}{\partial x_1}(x) \\\vdots \\\frac{\partial f}{\partial x_n}(x)\end{bmatrix}. ∇f(x)=⎣⎡​∂x1​∂f​(x)⋮∂xn​∂f​(x)​⎦⎤​.

The gradient and directional derivative are related via the formula

f′(x;d)=∇f(x)T ⁣d. f'(x;d) = \nabla f(x)^T\! d. f′(x;d)=∇f(x)Td.

If, for example, the direction ddd to be the canonical unit direction eie_iei​, then this formula reduces to

f′(x;ei)=∇f(x)T ⁣ei=[∇f(x))]i=∂f∂xi(x), f'(x;e_i) = \nabla f(x)^T\! e_i = [\nabla f(x))]_i = \frac{\partial f}{\partial x_i}(x), f′(x;ei​)=∇f(x)Tei​=[∇f(x))]i​=∂xi​∂f​(x),

which confirms the identity (1).

Linear approximation

The gradient of a continuously differentiable function fff (i.e., fff is differentiable at all xxx and ∇f\nabla f∇f is continuous) provides a local linear approximation of fff in the following sense:

f(x+d)=f(x)+∇f(x)T ⁣d+ο(∥d∥), f(x+d) = f(x) + \nabla f(x)^T\! d + \omicron(\| d\|), f(x+d)=f(x)+∇f(x)Td+ο(∥d∥),

The residual ο:R+→R\omicron:\mathbb R_+\to\mathbb Rο:R+​→R of the approximation decays faster than ∥d∥\| d\|∥d∥, i.e.,

lim⁡α→0+ο(α)/α=0.\lim_{α\to0+}\omicron(α)/α=0.α→0+lim​ο(α)/α=0.

Example

Fortunately, there are good computational tools that automatically produce reliable gradients. Consider the 2-dimensional Rosenbrock function and its gradient:

f(x)=(a−x1)2+b(x2−x12)2∇f(x)=[−2(a−x1)−4b(x2−x12)x12b(x2−x12)]\begin{aligned} f(x) &= (a - x_1)^2 + b(x_2 - x_1^2)^2 \\ \nabla f(x) &= \begin{bmatrix} -2(a-x_1)-4b(x_2-x_1^2)x_1 \\ 2b(x_2-x_1^2)\end{bmatrix} \end{aligned}f(x)∇f(x)​=(a−x1​)2+b(x2​−x12​)2=[−2(a−x1​)−4b(x2​−x12​)x1​2b(x2​−x12​)​]​

Here is the code for fff and its gradient:

a, b = 1, 100
f(x) = (a - x[1])^2 + b*(x[2] - x[1]^2)^2
∇f(x) = [-2(a - x[1]) - 4b*(x[2] - x[1]^2)*x[1] , 2b*(x[2] - x[1]^2) ]

Instead of computing gradients by hand, as we did above, we can use automatic differentiation, such as implemented in the package ForwardDiff, to compute these.

using ForwardDiff
∇fad(x) = ForwardDiff.gradient(f, x)

The function ∇fad returns the value of the gradient at x. Let's compare the hand-computed gradient ∇f against that automatically-computed gradient ∇fad at a random point:

x = rand(2) 
∇f(x) == ∇fad(x)
true

Calculus rules

We derive calculus rules for linear and quadratic functions, which appear often in optimization.

Linear functions

Let a∈Rna\in\mathbb R^na∈Rn. The linear function

f(x)=aT ⁣x=∑inaixi f(x) = a^T\! x = \sum_i^n a_i x_i f(x)=aTx=i∑n​ai​xi​

has the gradient ∇f(x)=a\nabla f(x) = a∇f(x)=a, and so the gradient is constant. Here's a small example:

a = collect(1:5)
ForwardDiff.gradient(x->a'x, rand(5))

Quadratic functions

Let A∈Rn×nA\in\mathbb R^{n\times n}A∈Rn×n be a square matrix. Consider the quadratic function

f(x)=12xT ⁣Ax. f(x) = \tfrac12 x^T\! A x. f(x)=21​xTAx.

One way to derive the gradient of this function is to write out the quadratic function making explicit all of the coefficients in AAA. Here's another approach that uses the product rule:

∇f(x)=12∇(xT ⁣a)+12∇(bT ⁣x),\begin{aligned} ∇f(x) = \tfrac12 ∇(x^T\! a) + \tfrac12 ∇(b^T\! x), \end{aligned}∇f(x)=21​∇(xTa)+21​∇(bTx),​

where a=Axa = Axa=Ax and b:=AT ⁣xb:=A^T\! xb:=ATx are held fixed when applying the gradient. Because each of the functions in the right-hand side of this sum is a linear function, we can apply the calculus rule for linear functions to deduce that

∇f(x)=12Ax+12AT ⁣x=12(A+AT ⁣)x. ∇f(x) = \tfrac12 Ax + \tfrac12 A^T\! x = \tfrac12 (A+A^T\!)x. ∇f(x)=21​Ax+21​ATx=21​(A+AT)x.

(Recall that AAA is square.) The matrix 12(A+AT ⁣)\tfrac12(A+A^T\!)21​(A+AT) is the symmetric part of AAA. If AAA is symmetric, i.e., A=AT ⁣A = A^T\!A=AT, then the gradient reduces to

∇f(x)=Ax. ∇f(x) = Ax. ∇f(x)=Ax.

But in optimization, we almost always assume that the matrix that defines the quadratic in (3) is symmetric because always xT ⁣Ax=12xT ⁣(A+AT ⁣)xx^T\! Ax = \tfrac12 x^T\!(A+A^T\!)xxTAx=21​xT(A+AT)x, and therefore we can instead work with the symmetric part of AAA.

f1(x)=∥x∥2andf2(x)=12∥x∥22. f_1(x) = \| x\|_2 \quad\text{and}\quad f_2(x) = \tfrac12\| x\|_2^2. f1​(x)=∥x∥2​andf2​(x)=21​∥x∥22​.

which isn't differentiable at the origin.

Visualizing gradients

Gradients can be understood geometrically in relation to the level-set of the function. The ααα-level set of a function f:Rn→Rf:\mathbb R^n\to\mathbb Rf:Rn→R is the set of points that have an equal or lower value at xxx:

[f≤α]={x∈Rn∣f(x)≤α}. [f≤α] = \{x\in\mathbb R^n\mid f(x)≤α\}. [f≤α]={x∈Rn∣f(x)≤α}.

Fix any xxx and consider the level set [f≤f(x)][f≤f(x)][f≤f(x)]. For any direction ddd that's either a descent direction for fff or a tangent direction for [f≤f(x)][f≤f(x)][f≤f(x)],

f′(x;d)=∇f(x)T ⁣d≤0, f'(x;d) = ∇f(x)^T\! d ≤ 0, f′(x;d)=∇f(x)Td≤0,

which implies that the gradient ∇f(x)\nabla f(x)∇f(x) is the outward normal to the level set.
